## 🚀 Project Overview

HeartSense-AI is an intelligent healthcare prediction system that uses an **Artificial Neural Network (ANN)** to predict the risk of heart disease based on patient medical attributes.

This project combines:

- 🧠 Deep Learning (ANN)
- ⚡ Flask REST API
- 🎨 Streamlit Interactive UI
- 📊 Data Visualization & EDA
- 🔍 Real-time Predictions

The goal is to demonstrate how AI can assist in early medical risk detection.

---

# 🧠 AI Model Architecture

The ANN model includes:

- Dense Neural Layers
- Batch Normalization
- Dropout Regularization
- Binary Classification Output
